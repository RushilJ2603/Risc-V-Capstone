# RISC-V Capstone Project

This repository contains the source code and documentation for my RISC-V Capstone Project. The project demonstrates the design, implementation, and testing of a RISC-V based system.

## Features

- Custom RISC-V processor core
- Assembly and C code examples
- Simulation and verification scripts
- Documentation and design files

## Getting Started

1. **Clone the repository:**
    ```bash
    git clone https://github.com/<your-username>/Risc-V-Capstone.git
    ```
2. **Install dependencies:**  
    Refer to the `requirements.txt` or relevant documentation for toolchain setup.

3. **Build and run simulations:**  
    See the `docs/` folder for instructions on building and running the project.

## Folder Structure

- `src/` - Source code for the processor and peripherals
- `tests/` - Testbenches and example programs
- `docs/` - Design documentation and setup guides
